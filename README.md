# hamster

Домашняя страница (англ. home page, разг. __хомяк__) — вымирающий вид веб-страниц. Как правило, размещены на бесплатном хостинге и сделаны либо в генераторе сайтов, либо свёрстаны в редакторе типа Front Page, а то и вообще в Microsoft Word (Сохранить как «Веб-страница»). На домашней странице содержится краткая биография автора, список его музыкальных/игровых/кино/футбольных/кулинарных предпочтений и фотография типа «я и моя сраная кошка». © Лукоморье
